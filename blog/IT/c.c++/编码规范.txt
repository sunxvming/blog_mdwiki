2.命名空间
  a.客户端使用 namespace star{} 命名空间包裹
  b.全局函数也封装于命名空间内
  c.禁止使用using namespace xx; 建议使用using namespace LORD::FOO;类型
  d.变量使用举例
    // 低效率 禁止使用
    for (int i=0; i<1000; ++i)
    {
        Foo f;
        f.DoSomething(i);
    }
    // 建议使用
    Foo f;
    for (int i=0; i<1000; ++i)
    {
        f.DoSomething(i);
    }
3.类
  a.仅仅有数据时使用struct,其他一律使用class
  c.成员变量初始化顺序,声明顺序对应
4.其他C++
  a.输入参数为值或者const引用
  b.输出参数为指针
  c.输入参数可以为const指针，禁止使用非const引用参数
  d.禁止使用RTTI(typeid、dynamic_cast等)
  e.使用c++类型转换，static_cast<>，建议不要使用int* y = (int*)x;  验证 B* b = (B*)c; B* b = static_cast<B*> c;
  f.使用前置自增自减，尤其迭代器和模板类型
  g.整数用0 实数用0.0 指针用NULL 字符串用'\0'
  h.使用sizeof(value)替代sizeof(type)
5.命名规定
    常量：大写加下划线
    文件名:同类名一致，一般一个类一个文件
    类名：大驼峰 class MyClass;
    成员变量：小驼峰，foo_ 下划线结尾
    成员函数：小驼峰
    变量名：小写加下划线
    普通函数：小写加下划线
